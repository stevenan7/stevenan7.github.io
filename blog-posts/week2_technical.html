Get Cultured, Ya Brute!

"Everything on a webpage is a box," said the wise man. If you don't have the option of meeting the wise man for a scone and latte, let me explain. The webpage you're reading this from is shaped like a box, and every element inside this webpage--buttons, pictures, menus, text-- are boxes interacting with each other inside the webpage. They each take up a certain length and width and are spaced between each other as if they are on a grid.
In order to define the space between different boxes, we can use the terms margin, border, and padding, If you look at the beige box this text is in, you can think of the text as the "content." The space around the content, and in this case, the beige space between the text and the edges of the beige box is defined as the padding. The space between the beige box and any other box/element (the header above, the beige box to the right) is defined as the margin, while the space between the padding and margin is defined as the border. Around this beige box there is no border because there is no designated space, i.e. a black line, between the beige box and the space around it.
