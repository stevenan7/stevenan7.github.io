Get Cultured, Ya Brute!

A hash is a convenient way to store keys and their corresponding values. Hashes can be created using Hash.new, and a default setting could be set as in weight = Hash.new(0). You can access values in hashes by using the keys for the corresponding value. For example, if you have a hash that is weights = {"Jared before subway" => 350, "Jared after subway" => 150, "Jared after getting rich" => 250}, you could call any of the values for Jared's weight at different times by calling weight["Jared after getting rich"] and 250 would be returned.
Hashes can be useful for storing information that is iterated throughout a program, rather than having to write out Jared's weight each time you want to call it. I'd say hashes in ruby are my second favorite type of hashes, after hash browns and before well, the kind you wouldn't want to smoke before coding!
